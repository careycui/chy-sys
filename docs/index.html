Normalize

normalize.css只是一个很小的css文件，为默认的HTML元素样式上提供了跨浏览器的高度一致性。相对于reset的简单粗暴，normalize.css则温柔许多。统一样式的同时保留可辨识性。

reset简单粗暴的通配符： 

* {
	margin: 0;
	padding: 0;
	border:0;
  }

ysui使用 normalize.css 。

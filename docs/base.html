基础设置 Base

1、盒子模型

YSUI 将所有元素的盒子模型设为border-box。 

*,
*:after,
*:before {
    -moz-box-sizing: border-box;
    -webkit-box-sizing: border-box;
    box-sizing: border-box;
}
注：使用通配符'*'是不好的。不过暂时没法解决这个问题/(ㄒoㄒ)/~~

2、字体

字体设置为 "Helvetica Neue", Helvetica, Arial, "Microsoft Yahei", "Hiragino Sans GB", sans-serif，基础字号为14px，基础行高 line-height:1.5，字体颜色 #333。

  • Helvetica Neue：Helvetica Neue为Helvetica的改善版本
  • Helvetica：被广泛使用的传奇般的西文字体 Mac
  • Arial：Win平台上默认的无衬线西文字体
  • Microsoft Yahei：微软雅黑，中文字符
  • Hiragino Sans GB：冬青黑体 Mac
由于英文字体中大多不包含中文，我们可以先进行英文字体的声明，这样不会影响到中文字体的选择，因此优先使用最优秀的英文字体，中文字体声明则紧随其次 
html {
  font-family: "Helvetica Neue", Helvetica, Arial, "Microsoft Yahei", "Hiragino Sans GB", sans-serif;
  /* 1 */
  line-height: 1.5;
  /* 2 */
  font-size: 14px;
  color: #333;
  -ms-text-size-adjust: 100%;
  /* 3 */
  -webkit-text-size-adjust: 100%;
  /* 3 */
  -webkit-font-smoothing: antialiased;
}

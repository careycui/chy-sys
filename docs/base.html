基础设置 Base

1、盒子模型

YSUI 将所有元素的盒子模型设为border-box。 

*,
*:after,
*:before {
    -moz-box-sizing: border-box;
    -webkit-box-sizing: border-box;
    box-sizing: border-box;
}
注：使用通配符'*'是不好的。不过暂时没法解决这个问题/(ㄒoㄒ)/~~

2、字体

字体设置为 Arial, Helvetica, "Microsoft Yahei", simsun, "Hiragino Sans GB", sans-serif，基础字号为14px，基础行高 line-height:1.5，字体颜色 #333。

  • Arial：英文字符 Mac&Win
  • Helvetica：英文字符 Mac
  • Microsoft Yahei：微软雅黑，中文字符
  • simsun：宋体，中文字符
  • Hiragino Sans GB：冬青黑体 Mac
html {
  font-family: "Microsoft Yahei", simsun, "Hiragino Sans GB", arial, sans-serif;
  /* 1 */
  line-height: 1.5;
  /* 2 */
  font-size: 14px;
  color: #333;
  -ms-text-size-adjust: 100%;
  /* 3 */
  -webkit-text-size-adjust: 100%;
  /* 3 */
  -webkit-font-smoothing: antialiased;
}

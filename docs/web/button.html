按钮 Button

1、基本使用

为按钮添加ys-btn-*样式，再设置相应的应用样式。

  • ys-btn - 基本按钮样式
  • ys-btn-lg 大号按钮样式
  • ys-btn-sm 小号按钮样式
  • ys-link-btn - 线框按钮样式
  • ys-link-btn-reverse - 反色线框按钮样式
  • ys-color-btn-primary - 彩色按钮样式 primary

1.1 基本按钮

四种样式按钮。

<button class="ys-btn">ys-btn</button>
<button class="ys-btn ys-link-btn">ys-link-btn</button>
<button class="ys-btn ys-link-btn-reverse">ys-link-btn</button>
<button class="ys-btn ys-color-btn-primary">ys-color-btn-primary</button>
1.2 按钮大小
<button class="ys-btn ys-color-btn-primary ys-btn-sm">小号按钮</button>
<button class="ys-btn ys-color-btn-primary">默认大小</button>
<button class="ys-btn ys-color-btn-primary ys-btn-lg">大号按钮</button>
    
<button class="ys-btn ys-link-btn ys-btn-sm">小号按钮</button>
<button class="ys-btn ys-link-btn">默认大小</button>
<button class="ys-btn ys-link-btn ys-btn-lg">大号按钮</button>
    
<button class="ys-btn ys-link-btn-reverse ys-btn-sm">小号按钮</button>
<button class="ys-btn ys-link-btn-reverse">默认大小</button>
<button class="ys-btn ys-link-btn-reverse ys-btn-lg">大号按钮</button>
    
<button class="ys-btn ys-btn-sm">小号按钮</button>
<button class="ys-btn">默认大小</button>
<button class="ys-btn ys-btn-lg">大号按钮</button>
1.3 Icon按钮

使用fontawsome图标。

<button class="ys-btn ys-color-btn-primary"><i class="fa fa-plus"></i>添加</button>
<button class="ys-btn ys-color-btn-primary"><i class="fa fa-spinner fa-spin"></i> Loading</button>
<button class="ys-btn ys-color-btn-primary"><i class="fa fa-shopping-cart"></i> 购物车</button>

2、按钮状态

2.1 禁用

禁用状态添加disabled类。

<button class="ys-btn disabled">ys-btn</button>
<button class="ys-btn ys-link-btn disabled">ys-link-btn</button>
<button class="ys-btn ys-link-btn-reverse disabled">ys-link-btn</button>
<button class="ys-btn ys-color-btn-primary disabled">ys-color-btn-primary</button>
2.2 激活

激活状态添加active类。

<button class="ys-btn active">ys-btn</button>
<button class="ys-btn ys-link-btn active">ys-link-btn</button>
<button class="ys-btn ys-link-btn-reverse active">ys-link-btn</button>
<button class="ys-btn ys-color-btn-primary active">ys-color-btn-primary</button>

网格 类Bootsrap Grid

1、基本

参考Bootstrap的网格规范，划分为12列网格。通过添加ys-cell-[列数]类来控制网格，通过添加ys-cell-[列数]-[tablet|phone]确定响应式显示。

控制元素
class 控制内容
ys-grid 网格的行或列需要包含在此内，赋予正确内外边距
ys-grid-row 包含一行的列
ys-cell-[列数] 0~12列数，为0时写为ys-cel，填满一行。当一行包含列大于12时另起一行。
ys-cell-[列数]-[tablet|phone] ys-grid width 1200px > 1199px
959px ~ 1199px
tablet 479px ~ 959px
phone < 479px
屏幕划分
  > 1199px 959px ~ 1200px 479px ~ 960px < 480px
ys-grid 宽 100% 100% 100% 100%
ys-fixed-grid 宽 1200px 960px 100% 100%
最大列数(超出的列100%) 12 12 8 4

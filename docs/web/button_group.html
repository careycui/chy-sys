按钮组 Button Group

1、分页

一个简单的分页样式，包含悬浮状态、激活状态以及页面跳转输入框。

/ 12页 
  
<div class="ys-page">
    <button class="ys-btn">上一页</button>
    <button class="ys-btn active">1</button>
    <button class="ys-btn">2</button>
    <button class="ys-btn">3</button>
    <button class="ys-btn">...</button>
    <button class="ys-btn">11</button>
    <button class="ys-btn">12</button>
    <div class="ys-page-nums">
        <input type="text" class="ys-field ys-field-inline" value="1">
        <span>/</span>
        <span>12页</span>
    </div>
    <button class="ys-btn">下一页</button>
</div>
